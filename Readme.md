## Getting Started

### Dependencies
mdflash is a self-contained, pure C code that stands tall only with GTK library! ğŸ°

### Installing
Set of steps for installation:

```
git clone https://github.com/yellow-footed-honeyguide/mdflash.git 
cd mdflash
mkdir build
cd build
meson ..
ninja
```

Now you can move nooks to /usr/local/bin/
```
sudo mv ./mdflash /usr/local/bin/
```



### Examples of usage
```
$ mdflash Readme.md
```
